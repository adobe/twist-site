    BaseLayoutComponent

    The base class of the "model representation of an item" within a VirtualScroll view. This class stores an item's position information and data: a model of the view, but not the actual view itself. An actual view (which would extend BaseViewComponent

    • you can use the @ViewComponent decorator for this) would be recycled and reused as the VirtualScroll view scrolls.
