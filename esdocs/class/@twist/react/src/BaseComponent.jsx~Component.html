    public get props: object source

    Accessor for props - this is necessary for rendering to work if you access props directly in the renderer. However, normal @Attribute accessors don't go through here (they access the internal [_props] instead), so that the component doesn't need to re-render if a prop that it doesn't use changes.

    We encourage using @Attribute rather than accessing props directly, for these performance reasons.
